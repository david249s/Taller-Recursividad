# Taller-Recursividad
David Santiago Martinez Muñoz

Este taller tiene como objetivo aplicar el concepto de recursividad  en programacion previamente visto en clase, mediante la solución de 14 diferentes problemas

Para ejecutarlo:
Tener instalado Java
Guardar el archivo
Abrir la terminal o consola
Compilar el programa
Ejecutar el programa
Ingresar los datos por consola
